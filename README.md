# deegame
Simple game made using D and DGame

Dependencies
============

Place the following dependencies in your project's bin directory

- libFLAC-8
- libfreetype-6
- libjpeg-9
- libmikmod-2
- libmodplug-1
- libogg-0
- libpng16-16
- libtiff-5
- libvorbis-0
- libvorbisfile-3
- libwebp-4
- SDL2
- SDL2_image
- SDL2_mixer
- SDL2_ttf
- smpeg2
- zlib1
- arial.ttf - Copied from your system's font directory
